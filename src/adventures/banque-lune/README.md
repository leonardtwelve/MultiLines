# Le Casse de la Banque Lune

Aventure cyberpunk lunaire pour 3 à 5 joueurs, ~35 min, ton malicieux.

## Pitch
2117. La Banque Lune — citadelle financière en orbite — abrite les coffres-forts privés de la haute société terrienne. Une équipe de spécialistes a une nuit pour s'y introduire. Mais l'un d'eux est un infiltré.

## Rôles asymétriques (à venir)
- **Hacker** : casse les serrures électroniques.
- **Ingénieur** : neutralise les systèmes de sécurité.
- **Acrobate** : franchit les pièces piégées.
- **Stratège** : connaît le plan complet.
- **Infiltré** : doit faire échouer le casse sans se faire repérer.

## Statut
Squelette en place — voir `index.ts` pour la scène placeholder. Le gameplay viendra dans les prochaines itérations.

## Structure du dossier
- `manifest.ts` — métadonnées (titre, joueurs, durée, ton).
- `index.ts` — implémentation du contrat `Adventure`.
- `scenes/` — scènes Phaser propres à l'aventure.
- `roles/` — définitions et logiques de rôles.
- `assets/` — sprites, sons, dialogues.
